1. Anchor on the team colour Default = FotMob primary. It's honest to the data and already looks right for ~80% of pairings.
2. Measure the pair in a perceptual space Convert both to OKLCH/LAB and compute ΔE. If ΔE (and hue gap) exceed a threshold, do nothing.
3. De-clash by rotating the AWAY hue Too close? Keep home, rotate away's hue and split lightness (one lighter, one darker) until they separate — never invent an off-brand colour, just bias within the family (red→crimson/maroon).
4. Guard the canvas Clamp lightness so neither blanket is near-black (Germany grey) or near-white (Senegal kit) against the base — nudge toward mid-lightness so both always glow.
5. Flag whisper only on a true clash If hue can't separate them enough, add a very low-contrast flag-band/star texture as a second channel (colourblind-safe). Off by default; a whisper, never stripes-as-noise.
6. Lifted plum-navy base Background #101625→#0b0f18 reverse-vignette (from v5). Gives dark colours an edge, adds gallery atmosphere, keeps blankets the star.
